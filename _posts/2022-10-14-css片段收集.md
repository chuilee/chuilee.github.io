#### 设置file input 鼠标手形

```scss
.uploader-input {
  position: relative;
  display: inline-block;
  line-height: 40px;
  padding: 0 10px;
  text-decoration: none;
  input {
    position: absolute;
    font-size: 100px; // 重要
    cursor: pointer;  // 重要
    top: 0;
    left: 0;
    right: 0;
    bottom: 0;
    opacity: 0;
  }
}
```
